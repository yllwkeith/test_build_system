### Настройка и запуск

#### Переменные окружения

build -- путь до файла с билдами, по умолчанию builds.yml

tasks -- путь до файла с сборками, по умолчанию tasks.yml

#### Запуск

локально

```
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8080
```

или через докер

```
docker build . -t test_build_system
docker run -p 8080:8080 test_build_system
```

### Запуск тестов

```
pytest tests.py
```
